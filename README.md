# HomeOn Energy Card

## Instalacja przez HACS
1. HACS -> Frontend.
2. Trzy kropki -> Custom repositories.
3. Dodaj repozytorium: https://github.com/maleikap/homeon-energy-card
4. Typ: Dashboard.
5. Zainstaluj HomeOn Energy Card.
